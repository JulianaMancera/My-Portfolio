# Juliana Mancera — Portfolio

Personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a cyberpunk-inspired design with animated backgrounds, scroll-reveal effects, a custom cursor, and a loading screen.

## Sections

- **Home** — Hero intro with animated title
- **About** — Background and education
- **Skills** — Languages, frameworks, databases, and tools
- **Projects** — Featured work
- **Achievements** — Certifications and recognitions
- **Contact** — Links and reach-out form
